唯一分解定理

任何一个大于1的自然数 N,如果N不为质数，那么N可以唯一分解成有限个质数的乘积N = P1 ^ a1 * P2 ^ a2 * P3 ^ a3 ... Pn ^ an，这里P1 < P2 < P3 <...< Pn且均为质数，其中指数ai是正整数。这样的分解称为 N 的标准分解式。
